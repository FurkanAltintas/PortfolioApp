# PortfolioApp
A simple portfolio project made for learning Dapper.

### Project Details
------------
- N Layered Architecture
- Dapper
- Asp.Net Core
- Asp Net Core Custom Cookie Auth.
- AutoMapper
- Dapper Contrib
- Fluent Validation
- Code First
- Dto (Data Transfer Object)
